## Work Code Examples (Dev Heroes)

### S3UnityClient
A Unity client for Amazon S3 handling async file upload, download 
and deletion. Features resumable downloads, MD5 checksum validation, 
chunked transfer, and robust error handling.

### SpriteLoader
A sprite loading and caching system using UnityWebRequest. Features 
thumbnail-first progressive loading, aspect ratio fitting, a loading 
queue, and automatic cache cleanup for memory management.
